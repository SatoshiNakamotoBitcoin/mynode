Instructions to run Balance of Satoshis
  1. Balance of Satoshis (bos) is a tool for viewing and interacting with lnd.
  2. To get started, you need access to the Linux terminal of your myNode device.
    • A guide to access the Linux terminal is available here - guide
  3. Once you have terminal access, run commands starting with 'bos'. For example:
    • bos peers
    • bos balance
    • bos utxos
    • bos report
    • bos help
  4. Enjoy Balance of Satoshis!
