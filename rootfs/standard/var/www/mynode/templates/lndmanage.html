Instructions to run lndmanage
  1. lndmanage is an advanced channel management tool for lnd.
  2. To get started, you need access to the Linux terminal of your myNode device.
    • A guide to access the Linux terminal is available here - guide
  3. Once you have terminal access, run commands starting with 'lndmanage'. For example:
    • lndmanage status
    • lndmanage listchannels
    • lndmanage rebalance ...
    • lndmanage circle ...
  4. Enjoy lndmanage!
